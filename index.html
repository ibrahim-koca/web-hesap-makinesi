M

Klavye Kısayolları

  • 0–9: Sayılar
  • +, -, *, /: Operatörler
  • . veya ,: Ondalık
  • (, ): Parantezler
  • Enter: Hesapla
  • Backspace: Son karakteri sil
  • Alt + C: Tam temizle
  • Alt + 1: Memory Clear (MC)
  • Alt + 2: Memory Recall (MR)
  • Alt + 3: Memory Add (M+)
  • Alt + 4: Memory Subtract (M-)
